# Architecture-Prompt-Library

## The Prompt Library
This is a free library of prompts to help architects build a strong architectural context from a simple description or an existing diagram. These are simply prompts I have created and found useful to integrate more perspectives into my work to build a solid context model I can then translate into many different sources. As always hallucinations are possible and you should verify all provided output.

### Business Architecture Prompts

These prompts are focused on helping you to describe the business value of your architecture, or further extend the LLM's context/understanding of the architecture.
- [The Architecture Vision & Creative Brief](prompts/architecture-vision-creative-brief.md)
- [The Framework Integration Interview](prompts/framework-integration-interview.md)

### Empathy and Design Thinking Prompts

This prompt (and future prompts as I want to work this area more) will help you define the personas of your system and can be used to map their overall satisfaction with your architecture. Use this to find missing features or ideation based on user needs.
- [The Empathy & UX Metric Engine](prompts/empathy-ux-metric-engine.md)

### Red Team Prompts

These prompts are used to pick at your architecture and find different holes you may not have considered.
- [The "Council of Critics" Red Team](prompts/council-of-critics.md)
- [The "Black Swan" Gap Finder](prompts/black-swan-gap-finder.md)

### Translation/Output Prompts

These prompts are used once you feel you have a well defined context to output that to various mermaid/markdown (or other) constructs. More to come here as well.
- [The Iterative Builder (Visualizer & Documenter)](prompts/iterative-builder.md)
- [SparxEA Script Creation](prompts/sparxea-script-creation.md)
